= Knife Slicehost
This is the official Opscode Knife plugin for Slicehost[http://www.slicehost.com/]
This plugin gives knife the ability to create, bootstrap, and manage Slicehost
slices.

== Installation
This plugin is distributed as a Ruby Gem. To install it, run:
  gem install knife-slicehost
Depending on your system configuration, you may need to run this command with
root privileges.
